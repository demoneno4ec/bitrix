### Здесь идиологически будут создаваться какой либо иснтрументарий. 
`который можно будет использовать на CMS Bitrix, и не только`

1. ***CLass*** - Содержит мои классы с autoloader для подключения классов и использования их внутри проекта
2. ***Components*** - Как только перепишу свои компоненты на D7, если до этого дойдет, тут будут и они
3. ***Templates*** - Шаблоны, которые можно переиспользовать.
